如何将本地文件上传到云服务器呢?

移动到本地文件目录下

cd file

scp index.css root@云服务器公网IP地址:服务器文件目录

回车输入密码就可以上传了

移动整个文件目录到云服务器指定目录

scp -r images root@云服务器公网IP地址:服务器文件目录

移动多个文件到云服务器指定目录

scp index.html index.css project1.html project2.html root@139.196.40.14:服务器文件目录
多个文件名之间用空格分开。
